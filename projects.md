<p class="lead">Things I've built and written about.</p>

A selection of public work — open source libraries, tools, and teaching material.
Most of it is in Go. Source for everything is on [GitHub](https://github.com/nyelonong).

## Featured

- **[nyelonong.github.io](https://github.com/nyelonong/nyelonong.github.io)** —
  this blog itself. Plain Jekyll, no plugins, no third-party JavaScript or
  fonts.
- **[galdr](https://github.com/nyelonong/galdr)** — an engineering methodology for
  Claude Code: routed requests, wave-based TDD plans, evidence gates, and durable
  memory. Published as a public Claude plugin.
- **[nsqpause](https://github.com/nyelonong/nsqpause)** — pause and unpause NSQ
  topics and channels. My most-used utility.

## Go libraries & tools

- **[nsqsink](https://github.com/nyelonong/nsqsink)** — read from NSQ and write to
  an HTTP endpoint.
- **[httpclient](https://github.com/nyelonong/httpclient)** — an easy-to-use HTTP
  client for Go.
- **[gommon](https://github.com/nyelonong/gommon)** — common Go utilities.
- **[checkup](https://github.com/nyelonong/checkup)** — a healthcheck library.
- **[quran](https://github.com/nyelonong/quran)** — the Qur'an, in your terminal.
